# Mi Primera App

Aplicación Android creada con Kotlin que permite procesar texto con alertas interactivas.

## Características

- Caja de Texto (EditText)
- Label (TextView)
- Botón
- Alerta Pop-up

## Requisitos

- Android Studio
- Android SDK API 34
- Kotlin 1.9.20
- Java 11

## Instalación

1. Clona este repositorio
2. Abre el proyecto en Android Studio
3. Sincroniza el proyecto con Gradle
4. Conecta un dispositivo o emulador
5. Presiona Run

## Uso

1. Ingresa texto en la caja
2. Presiona "Procesar Texto"
3. El texto aparece en el label
4. Se muestra una alerta con el mensaje
